1. Download and paste the folder in "C:\xampp\htdocs
2. Open xampp and create a new database "cs_database"
3. Click import and select the databse in "C:\xampp\htdocs\Campus-Connect-Web-App\database\cs_database.sql"
4. Open "C:\xampp\mysql\bin\my.ini" or "C:\xampp\mysql\bin\my" and change max_packet_size to 1G
5. Open web browser and paste "http://localhost/Campus-Connect-Web-App/index.php"
